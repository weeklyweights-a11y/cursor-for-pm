Feedback to analyze:
---
{feedback_content}
---

Respond with a single JSON object with these fields only:
- pain_point (string): What the customer is struggling with. One sentence.
- topic (string): Category. One or two words. Examples: search, authentication, performance, onboarding, pricing, mobile, integrations.
- related_feature (string): Which existing or requested feature this relates to.
- is_existing_feature (boolean): true if about an existing feature, false if a request for something new.
- feature_gap (string or null): What is missing or broken. One sentence. Null if no clear gap.
- urgency (string): One of: low, medium, high, critical. Based on language intensity.
- sentiment (string): One of: positive, neutral, negative.
- verbatim_quote (string): The most important sentence or phrase from the feedback. Copy exactly.
- confidence (number): Your confidence in this extraction, 0.0 to 1.0.
